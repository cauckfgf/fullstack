京航安斯里兰卡班达拉奈克国际机场项目部：严格把控技术交底

京航安斯里兰卡班达拉奈克国际机场项目部为确保工程质量，严格把控施工过程中技术交底要求，抓好每一道施工环节，做好施工资料填写，确保资料与实地情况无差异。根据工作区域涉及飞机活动频繁地区多得实际情况，项目部加大现场督促力度，确保和运营单位及时沟通，周密检查车辆车况，杜绝有故障隐患的车辆进入现场。

京航安马尔代夫项目部：确保安全施工，精细组织施工

现在，正值马尔代夫季节转换时节，雨水较多，容易滋生蚊虫，诱发登革热等疾病。项目部注重对宿舍的卫生清理工作，保持了室内清洁。同时各工作面做好人员、材料和设备的防风防雨工作，加强雨后安全检查。各联络道口钻孔取芯时注意路面清洁，对已经损坏的沥青块进行清理。加强井接管部位的凿毛处理。电缆敷设时注意保护电缆，严禁损伤电缆外皮。基坑开挖工作中时注意基底夯实，土方回填时注意夯实。钢筋绑扎时注意保护层厚度，注意混凝土制品养护。混凝土浇筑注意提前准备防雨措施，保证浇筑质量。南灯光站施工时加强 PPE 管理，佩带安全帽，穿反光背心，防止高空坠落物，接地铜排及端子箱制作，注意施工工序影像资料的留存。跑道及滑行道作业时加强管理，严格遵守机场空侧区管理要求，各作业面车辆及设备对已经损坏或遗失的警示旗进行换新，严格遵守新跑道行驶路线，车辆在社会路行驶注意交通安全。

公司简介

北京京航安机场工程有限公司成立于2002年，前身是北京军区空军所属军办企业，2017年加入远东智慧能源，成为A股上市公司。 公司是首批取得国家民航局颁发的机场施工许可证的单位，是军民航机场专业工程建设领域的知名企业，是智慧机场系统解决方案服务商。 公司持有机场目视助航工程专业承包壹级、民航空管工程及机场弱电系统工程专业承包壹级、机场场道工程专业承包贰级、建筑机电安装工程专业承包贰级、机场场道工程专业承包贰级、电子与智能化工程专业承包贰级等工程资质，是目前国内同时具备机场目视助航工程专业承包壹级、民航空管工程及机场弱电系统工程专业承包壹级双重资质的四家单位之一。公司取得了质量管理体系、环境管理体系和职业健康安全体系认证，获得高新技术企业认证，是银行AAA级信用企业。 公司现有注册建造师、注册造价师、高级工程师、工程师等专业技术人才300余名。拥有先进的施工机械设备、仪器仪表等配套技术装备，具有较强的深化设计、技术研究和工艺创新能力，具备在4F飞行等级机场和不停航条件下组织工程建设的丰富经验和雄厚技术实力。 公司始终贯彻“做诚信企业，创精品工程”的经营理念，秉承“用户满意、持续创新、追求精品”的质量方针，坚持“认真、细致、高标准”的工作作风，弘扬“诚实、敬业、创新、敢为人先”的企业精神，专心专注于机场领域工程，努力打造“经营管理规范、施工工艺精湛、技术实力领先”的京航安品牌，赢得了客户的广泛赞誉。 公司成立以来，承建了国内外120多个军民航机场的500多个项目，先后荣获“中国建设工程鲁班奖”、“国家优质工程银质奖”、“中国市政工程金杯奖”、“市政工程金奖”、“扬子杯”、“金陵杯”等多项荣誉。 展望未来发展，公司将继续围绕“市场有竞争力，思想有凝聚力，工作有执行力，行业有影响力”的企业目标，不忘初心、砥砺奋进，不断提升企业核心竞争力，在激烈的市场竞争条件下实现企业的可持续发展，致力于成为全球领先的智慧机场服务商。
